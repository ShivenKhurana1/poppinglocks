# Poppinglocks!

Basically Pop the lock but browser based ahah

## How to Play

- Press **Spacebar** to start
- That white dot.. thats you :O
- Press **Spacebar** when the dot overlaps the red target
- you get it now don' cha :D

## Features
- Screen shake on miss
- Score bounce animation
- Animated padlock shackle
- Speed increases with each successful hit (duh)

## Files

- `index.html` — genuinely everything else
- `style.css` — makes it pretty
- `script.js` — logic

## Running Locally

Open `index.html` ... nothing much brochacho
